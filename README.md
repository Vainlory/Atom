## 安装

目前仅有Qt得源代码及工程还有项目所配套得图形库,如需查看效果请自行编译 

*有尝试过编译成release版，但是对于QT的文件的层次关系目前还不熟练，后续的release版会之后上传*

Qt环境是 Desktop Qt 5.13.2  MinGW-64bit

Mysql 是 mysql-8.0.21-winx64

*我采用了DMS数据库模式，将相关信息存在阿里云上的数据库中，可以自建本地数据库*
