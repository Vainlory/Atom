## 项目背景

某日脑子一抽，于是开始学习 qt 并着手写这个项目，没啥深层次原因 

## 安装

目前仅有Qt得源代码及工程还有项目所配套得图形库,如需查看效果请自行编译 

*有尝试过编译成release版，但是对于QT的文件的层次关系目前还不熟练，后续的release版会之后上传*

Qt环境是 Desktop Qt 5.12.9  Windows

## 使用

请根据日常生活中使用各类文本编辑器*(eg.  evernote , notepad )*得经验来使用他。
